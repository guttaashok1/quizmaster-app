Job Profiles

Save your resume, job description, and supporting context together. Load any profile on the interview page to start a session in one click.

🗄️
Profiles are stored locally on this device only. To sync profiles across any device after login, connect a free PostgreSQL database. Setup guide →
  1. Create a free database at neon.tech or supabase.com
  2. Copy your connection string (starts with postgres:// or postgresql://)
  3. In Vercel → your project → Settings → Environment Variables → add DATABASE_URL
  4. Click Redeploy — profiles will sync automatically from any device
