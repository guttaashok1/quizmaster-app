Supabase + Stripe Setup
  1. Go to supabase.com → create project → copy URL and anon key from Project Settings → API
  2. Go to stripe.com → create account → get publishable key and secret key from Developers → API keys
  3. In Vercel: Settings → Environment Variables → add SUPABASE_URL, SUPABASE_ANON_KEY, STRIPE_PUBLISHABLE_KEY, STRIPE_SECRET_KEY
  4. Redeploy from Vercel dashboard — Stripe webhooks and Supabase user sync will be active
